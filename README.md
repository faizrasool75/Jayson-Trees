# Jayson Trees

A modern, responsive website for Jason's Tree Service built with React, Vite, and TailwindCSS.

## Features

- Modern and responsive UI/UX design
- React-based single-page application
- Multiple pages: Home, Services, About, Gallery, Contact, and Why Choose Us
- Interactive components and smooth animations
- Mobile-friendly navigation
- Contact form integration
- Client reviews and testimonials
- Featured work gallery

## Tech Stack

- **React** - UI library
- **Vite** - Build tool and dev server
- **TailwindCSS** - Utility-first CSS framework
- **React Router** - Client-side routing

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/faizrasool75/Jayson-Trees.git
cd Jayson-Trees
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
